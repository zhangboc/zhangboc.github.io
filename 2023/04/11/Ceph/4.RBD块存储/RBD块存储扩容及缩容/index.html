RBD存储扩容

查看需要扩容的镜像

[root@node-1 data]# rbd ls ceph-demo 
rbd-demo.img

查看需要扩容的镜像

[root@node-1 data]# rbd -p ceph-demo info rbd-demo.img 
rbd image 'rbd-demo.img':
    size 10 GiB in 2560 objects
    order 22 (4 MiB objects)
    snapshot_count: 0
    id: 11e24b04cf01
    block_name_prefix: rbd_data.11e24b04cf01
    format: 2
    features: layering
    op_features: 
    flags: 
    create_timestamp: Sat Mar  4 02:03:28 2023
    access_timestamp: Sat Mar  4 02:03:28 2023
    modify_timestamp: Sat Mar  4 02:03:28 2023

使用rbd resize进行扩容帮助

[root@node-1 data]# rbd help resize  #使用rbd resize进行扩容
usage: rbd resize [--pool <pool>] [--namespace <namespace>] 
                  [--image <image>] --size <size> [--allow-shrink] 
                  [--no-progress] 
                  <image-spec> 

Resize (expand or shrink) image.

Positional arguments
  <image-spec>         image specification
                       (example: [<pool-name>/[<namespace>/]]<image-name>)

Optional arguments
  -p [ --pool ] arg    pool name
  --namespace arg      namespace name
  --image arg          image name
  -s [ --size ] arg    image size (in M/G/T) [default: M]
  --allow-shrink       permit shrinking  #缩容操作，不建议，可能会导致数据丢失
  --no-progress        disable progress output

将ceph-demo池下的rbd-demo.img镜像扩容到20G

[root@node-1 data]#  rbd resize ceph-demo/rbd-demo.img --size 20G 
Resizing image: 100% complete...done.
[root@node-1 data]# rbd -p ceph-demo info rbd-demo.img #查看已经扩容到20G了
rbd image 'rbd-demo.img':
    size 20 GiB in 5120 objects
    order 22 (4 MiB objects)
